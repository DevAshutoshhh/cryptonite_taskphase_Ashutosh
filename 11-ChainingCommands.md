## 2-Your First Shell Script

We can use any text editors like 
-Vim:

Open file: vim filename

Insert text: Press i

Save and exit: Press Esc, type :wq, hit Enter

-Nano:

Open file: nano filename

Type commands

Save: Ctrl + O, Enter

Exit: Ctrl + X

I used vim 
Solution -
 `vim x.sh`<br>
 `i` to input<br>
 `:wq` to save and quit<br>
 `bash x.sh`<br>
